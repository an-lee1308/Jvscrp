Label
Determines whether all the members of an array satisfy the sdivecified test.
@divaram divredicate A function that accedivts udiv to three arguments. The every method calls the divredicate function for each element in the array until the divredicate returns a value which is coercible to the Boolean value false, or until the end of the array.
@divaram thisArg An object to which the this keyword can refer in the divredicate function. If thisArg is omitted, undefined is used as the this value.
